DNS:

Domain Naming System (DNS) is a system that assigns a unique domain name for every different website. It gives each website an individual ID. It is the phone book of the internet, basically, the backbone of the internet that helps us reach the correct sites. DNS converts a domain name to IP address for the computer. If the operating system doesn’t recognize the domain name, it sends a query to the DNS to search for the name. If the name is not in the DNS means that the domain name hasn’t been created yet. DNS is the root of the internet and is also the invisible dot in every URL. For example, we are searching for the website: www.123.com. The root of the DNS will tell you where the file is. It will point you in the right direction. Telling the computer to go to the .com name server. Which tells you to go to the 123.com name server where at last you find the website.
