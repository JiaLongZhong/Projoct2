HDomain names and ICANN:

The Internet Corporation for Assigned Names and Numbers (ICANN) is the organization that certifies Domain Name to ensure that every address is unique and valid. Before ICANN there was only one-person name Jon Postel that distribute Internet-address and manage the entire domain name system. But since the Internet is used world-wide, it became problematic for a single person to keep track and distribute Domain Name, which cause an organization to form. The purpose of the Organization was to fit the creativity and speed of the Internet. Basically, a safely-net for the internet, a place where policy for the internet is created. That origination is called the ICANN, a non-government control organization that create policies that keep the internet running.

Domain name was created to make Internet protocols (IP addresses) more user-friendly. IP addresses is a set of number that is assign to every device that used the web. IP addresses tell the server the exact location of the device in the web and help route information throughout the web. The problem with IP addresses is that it is made out with numbers, which isn’t name friendly. Computer use IP addresses to communicate, but human use domain name to communicate. Basically, it is human nature to use name instead of number to identify anything.

The Domain Name Server (DNS) is what translate the domain name to IP address for the computer to communicate. The process of going to a website, start by a human typing in the domain name in the URL. The domain name then get transfer to the DNS, which convert the name to an IP addresses for the computer to find the website the associated with the addresses. Once the computer finds the website, it will display it on the computer screen. Allowing the user to access the website.
