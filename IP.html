IP Addresses, Packets and Routing:

An IP address is a numerical label assigned to each device, founded inside the IP package, and used by the router to connect to the network. IP addresses are usually four number that range from 0 to 255 separated by three dots. For instance: 125.65.67.123 is an IP address. These number are then converted to binary for the computer to use. Each IP address is 32 bits, which mean 32 one or zero. Devices need IP addresses to communicate with other over a network. This form of transport is like a mailing system. Where the IP address is the address of the network world. Unlike Mac address, IP address is not globally unique because router tend to reuse IP address, but within a router every IP address is different, which mean every device within the area is different.

IP Packets is like an email. It contain information that you want to send (data) with other information that help get the data to the destination (headers). IP header contain a minimum of 20 bytes in size. It contain a version field that tell you the IP address version (4 or 6). The header length field define the length of the IP headers. It also contain the Differentiated Services (DS) field. A packet field that identify the length of the packet. Then come the three-fragmentation field. A time to live field that prevent routing loop, a protocol flied identify what in the data. Header Checksum identify if data was corrupted during the process. The source IP address, which is the sender device IP address and the destination IP address, which is where you want to send your data to.

Once the IP packet is filled, it is sent to the router. The router is basically a transportation device. It receive the information (data) and transport to the destination. If the destination is out of reach in this router it pass it on to the next appropriated router. Router have Network address that tell itself where to send the data. It see the destination IP address as the destination and if no such destination is found it send it back to the source IP address. It is an online version of a mailing system.
